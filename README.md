# 📚 Система записи на курсы дополнительного профессионального образования «Корочки.есть»

Портал для записи пользователей на онлайн-курсы с возможностью оставлять отзывы и управлять заявками через панель администратора.

## 🚀 Основной функционал

- Регистрация и авторизация пользователей (с хэшированием паролей)
- Создание, просмотр и отслеживание заявок на обучение
- Оставление отзывов к курсам
- Административная панель для управления статусами заявок
- Валидация данных на клиенте и сервере
- Полностью асинхронное взаимодействие (AJAX / Fetch)

## 🛠️ Технологии

- **Backend**: Node.js + Express.js
- **Database**: PostgreSQL
- **Frontend**: HTML5, CSS3 (адаптив), JavaScript (Fetch API)
- **Безопасность**: bcrypt для хэширования паролей
